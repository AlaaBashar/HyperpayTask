Hi GitHub community,<br />
I made this app and I hope it would have your attention,<br />
It is just simple app to make short link for any link you insert it and stored as items inside the app via sqLite database,<br />
I use the the firebase to authenticate the email.

User Features:<br />

- Login & create new account in this app.<br />
- You can Upload your profile image & cover and you can Edit it in another time.<br />
- You can add & remove any Link to history List.<br />

Technical Features:<br />

- The code completely follow the roles of clean code & architecture.<br />
- I used sqLite as a Datebase to stored.<br />
- I used a server to make short Link (Request & Response).<br />
- At this point time I used to Provider & Bloc because these tools are more simple from the other.<br />
